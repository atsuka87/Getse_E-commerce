# 🛒 Getse Elektronik (DewiElektro) - E-Commerce Platform

Getse Elektronik (DewiElektro) adalah platform e-commerce modern yang dirancang khusus untuk penjualan produk elektronik berkualitas tinggi seperti smartphone, laptop, perangkat audio, smart TV, dan aksesoris elektronik lainnya. 

Proyek ini dibangun menggunakan teknologi terbaru **Laravel 12**, **Tailwind CSS (Vite)**, **Alpine.js**, dan **Laravel Breeze** untuk memberikan pengalaman berbelanja yang cepat, aman, dan responsif.

---

## ✨ Fitur Utama
*   **Katalog Produk Premium:** Tampilan grid produk modern, lencana diskon, stok barang, kondisi barang, dan spesifikasi lengkap untuk setiap produk.
*   **Kategori & Brand:** Navigasi cepat dengan filter berdasarkan kategori (misalnya Laptop, Handphone, Audio) dan brand terkemuka (Samsung, Apple, ASUS, Sony, dll.).
*   **Sistem Garansi Terintegrasi:** Menampilkan garansi resmi dan garansi toko secara jelas pada setiap detail produk.
*   **Manajemen Akun Terintegrasi:** Autentikasi aman untuk Administrator dan Pelanggan (Customer).
*   **Pengaturan Sosial & Toko Dinamis:** Pengaturan nomor WhatsApp, bank transfer, alamat fisik, dan sosial media yang dikonfigurasi melalui database.

---

## 🛠️ Stack Teknologi
*   **Backend:** PHP >= 8.2 & Laravel 12.x Framework
*   **Frontend Compiler:** Vite 7.x
*   **CSS Styling:** Tailwind CSS 4.x
*   **JavaScript Reactive:** Alpine.js 3.x
*   **Autentikasi & Scaffolding:** Laravel Breeze

## 🚀 Cara Menjalankan Website di Komputer Lokal

Kami telah menyediakan panduan langkah demi langkah yang sangat detail untuk memasang dan menjalankan website ini secara lokal. Silakan klik tautan di bawah ini untuk melihat panduan lengkap dalam Bahasa Indonesia:

👉 **[BACA PANDUAN CARA MENJALANKAN WEBSITE (CARA_MENJALANKAN.md)](CARA_MENJALANKAN.md)**

### Ringkasan Cepat:
1. Buat database MySQL bernama `dewi_ecommerce`.
2. Jalankan perintah setup otomatis:
   ```bash
   composer run setup
   ```
3. Jalankan seeding untuk mengisi data demo (Admin, Customer, Produk):
   ```bash
   php artisan db:seed
   ```
4. Jalankan website Anda dalam mode development:
   ```bash
   composer run dev
   ```
5. Buka **[http://localhost:8000](http://localhost:8000)** di browser Anda.
